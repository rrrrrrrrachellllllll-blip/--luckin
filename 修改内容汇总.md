# 商品洞察看板修改内容汇总

更新时间：2026-07-20

## 1. 数据看板结构调整

- 将数据看板从单一长页面整理为多个可进入的详情面板，减少主看板纵向堆叠。
- 在数据看板中保留概览入口，具体数据和图表进入详情界面查看。
- 数据面板已覆盖：预售数据、指南针取数、生意经数据、抖音整体数据。

## 2. 预售 / 指南针 / 生意经独立面板

- 预售、指南针、生意经分别作为数据看板下的独立面板入口。
- 点击对应面板后进入具体数据和可视化图表区域。
- 详情界面关闭方式改为必须点击右上角 X 号，避免误点空白区域导致退出。
- 面板打开增加点击动画反馈，让入口操作更明确。

## 3. 日历筛选功能

- 在对应数据板块加入年月日选择能力。
- 用户可以选择具体日期查看当天数据，减少多日数据同时堆叠造成的拥挤。
- 日历筛选用于帮助图表和数据表聚焦到单日口径。

## 4. 图表边缘显示修复

- 排查并修复图表最边缘数据、坐标轴标签、tooltip 可能显示不全的问题。
- 图表配置统一增强安全边距：tooltip 限制在视口内、文字自动换行、grid 包含标签。
- 对相关图表增加测试覆盖，避免后续再出现边缘信息被裁切。

## 5. 抖音整体看板调整

- 删除“抖音整体原始数据”板块。
- 主数据看板中的“抖音整体”只保留总览入口和关键指标，不再把下钻看板直接铺在下方。
- 点击“抖音整体”后进入独立的“抖音整体数据看板”详情界面。
- 新详情界面内展示“抖音整体下钻看板”，包含 POI 与直播板块的下钻数据和图表。
- 详情界面仍然保持只通过 X 号关闭，避免误关闭。
